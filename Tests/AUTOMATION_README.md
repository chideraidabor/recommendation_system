# 🤖 Invoice Form Automation Testing

## Quick Setup

### 1. Install Required Packages
```powershell
pip install -r automation_requirements.txt
```

### 2. Start Flask Application
```powershell
python app.py
```
(Keep this terminal running)

### 3. Run Automation Test
```powershell
cd Tests
python simple_automation_test.py
```

✅ Invoice number auto-generation  
✅ Date validation
